How to Use
  1. Change Theme: Click on "Settings" in the top-right corner
  2. Select Theme: Choose from 7 beautiful themes (Purple, Blue Ocean, Teal Marine, Emerald, Indigo, Rose, Amber)
  3. Watch Colors Update: All colors on this page update instantly!
  4. Automatic Save: Your theme preference is saved automatically
  5. Persistent Across Pages: Your theme choice persists across all pages in the system
  6. View Console: Open browser console (F12) to see theme change logs

The theme system uses CSS custom properties and JavaScript event listeners to provide real-time color updates throughout the entire application.
